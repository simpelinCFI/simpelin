# Select/Dropdown
Class inv-select merupakan wrapper yang membungkus select supaya styling bisa dilakukan dengan lebih mudah

```html
<div class="inv-select">
    <select>
        <option>Select dropdown</option>
        <option>With options</option>
    </select>
 </div>
```
